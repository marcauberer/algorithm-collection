# Linear Search
**Algorithm type**: Search algorithm <br>
**Condition**: No condition <br>
**Time complexity**: O(n)

## Pseudocode
```
function search(inputValues: List, item: Integer)
   i ← 0
   while inputValues[i] != item
      i ← i + 1
   return inputValues[i], i
end function
```
